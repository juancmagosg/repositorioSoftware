Software para crear Presentaciones Educativas Digitales

Genially

Es una herramienta en línea que permite crear diversos materiales visuales e interactivos de forma sencilla, mediante el uso de plantillas editables. Lo anterior, incluye presentaciones, infografías, gamificación, etc.


Es de tipo freemium, se puede utilizar de manera gratuita, pero también se puede pagar por acceder a las opciones avanzadas.

Prezi

Es un software en la nube que permite crear presentaciones sobre un documento virtual, destacando de forma particular el uso que hace del zoom para acercar o alejar elementos relevantes dentro de la presentación.


Es también de tipo freemium, es decir, que cualquier persona puede utilizarlo de forma gratuita, pero al realizar un pago se accede a un mayor número de herramientas.
